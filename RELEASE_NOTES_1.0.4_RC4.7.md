# CoachSplit 1.0.4 RC4.7

## Bildexport

- Bei aktivierter Altersklassenwertung werden Altersklassen als eigene
  Überschriften mit den zugehörigen Athleten dargestellt.
- Bei `Schießstand aus` enthält die jeweilige Spalte:
  - Schieß-/Abschnittszeit
  - gesamte Strafzeit dieses Schießens
  - weißen Fehlerkreis mit der Fehleranzahl
- Die Strafzeit wird nur angezeigt, wenn sie größer als null ist.
- Der Anschlag bleibt Teil des Spaltenkopfs.

## HTML-Export

Der HTML-Export ist eine browserbasierte Ergebnisdatei. Er kann lokal geöffnet,
per E-Mail oder Cloud geteilt und auf Handy oder Desktop ohne Spezialsoftware
angesehen werden.
