# Regressionstest RC4.3

1. Wettkampfuhr auf eine deutlich von der Systemzeit abweichende Zeit kalibrieren.
2. ersten Start etwa eine Minute nach der kalibrierten Zeit setzen.
3. einige Sekunden warten und `Zum Start` öffnen.
4. AutoStart aktivieren.
5. Der sichtbare Countdown muss der verbleibenden Zeit auf der Wettkampfuhr entsprechen.
6. Bei 00:00 muss der Athlet automatisch starten.
7. Ohne Kalibrierung muss weiterhin die lokale Gerätezeit verwendet werden.
8. Manueller Start und Messwerterfassung unverändert testen.
