# CoachSplit 1.0.6 RC6.3

Erster einfacher Mehrgeräte-Durchstich:

- Bewerb auf einem Administratorgerät freigeben
- messpunktspezifische Verbindungscodes erzeugen
- zweites Smartphone als Helfer verbinden
- Bewerbsdaten auf das Helfergerät übernehmen
- Messereignisse lokal zuerst speichern und danach synchronisieren
- Rollen und Messpunktberechtigungen serverseitig validieren
- doppelte Übertragungen idempotent behandeln
- Netzunterbrechungen über eine persistente lokale Outbox überstehen
- fremde Ereignisse automatisch einlesen und in der bestehenden Ansicht darstellen

RC6.3 ist für einen kontrollierten Test im lokalen Trainingsnetz vorgesehen.
